# AI Customer Intelligence Hub

A smart CRM dashboard that:
- Detects high-risk customers using AI
- Prioritizes support tickets
- Simulates WooCommerce integration
- Provides real-time customer insights

## Features
- Live incoming customer feed
- Risk-based sorting
- AI insights
- Action system (Reply, Resolve, Escalate)

## Tech Used
- HTML, CSS, JavaScript
- Simulated WooCommerce API
